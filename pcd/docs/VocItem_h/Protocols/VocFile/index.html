VocFile

Extends Protocol:
NSObject
Declared In:
VocService.h

Introduction

Represents a file.


Methods

-getFileType

Identify the type of file.

getFileType

Identify the type of file.

- (VocFileType) getFileType;
Return Value

VocFileType SDK assigns a type for each file internally. This type needs to be known by the client because main file can have sub files. For example: HLS has main m3u8 file, index file, segments, keys etc.

Properties

attributes

The file attributes provided during ingest.

autoDownload

During ingest the customer can indicate whether a particular file needs to be autodownloaded. This is applicable only for files other than VocMainContentType files and main thumbnail.

bytesDownloaded

Size in bytes for the file stored in cache. This property can be used to track the download progress while the file is getting downloaded.

fileGroupType

In addition to main files, SDK can download file groups such as subtitles and thumbnails. This property is used to identify which file group the file belongs to. The type information is provided during content ingest.

localPath

local device path to access the file.

mimeType

The file mimetype provided during ingest. SDK does not change it.

quality

The file quality provided during ingest. It ranges from 0 to 10.

size

The file size provided during ingest. Except for HLS and DASH content, the size will be corrected during download.

state

The primary aim for the SDK is to cache the right file. This property indicates the downloading state of the file.

url

The url that will be used to download the file.
