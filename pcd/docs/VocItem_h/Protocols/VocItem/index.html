Introduction

Represents a content item.


Methods

-deleteFiles:

Delete content in the file system but not the metadata.

-deleteFilesWithCompletion:completion:

Delete content in the file system but not the metadata.

-deleteItem

Delete the item from cache.

-lock

Use this method to signal SDK that the item should not be evicted from the cache or modified in a way that will prevent the user from consuming/viewing it.

-unlock

Use this method to unlock an item that was previously locked. Calling it on unlocked item has no effect.

        completion:(nullable void (^)(NSError * __nullable))completion;
Parameters
deleteThumbnail

If NO, delete all files associated with the item except thumbnail

completion

The completion block executes once deletion is completed. If deletion fails, it returns non nil error.

deleteItem

Delete the item from cache. 

- (void)deleteItem;

lock

Use this method to signal SDK that the item should not be evicted from the cache or modified in a way that will prevent the user from consuming/viewing it.

- (BOOL)lock;
Discussion

It is not recommended to lock more than couple of items since this will lock items in cache and less space will be available for pre-fetching. Use this method for items that the user started interacting with. As soon as possible unlock the item.

See

VocServiceDelegate::itemsEvicted:

unlock

Use this method to unlock an item that was previously locked. Calling it on unlocked item has no effect. 

- (void)unlock;

Properties

bytesDownloaded

Size in bytes for all things stored in cache belonging to this content item

categories

The categories associated with the content item.

contentId

Unique ID assigned for the content during ingest. If SDK has PCD capabilities, this property represents the content_unique_id during ingest.

downloadBehavior

Each item can have separate download behavior and it takes value from VOCItemDownloadBehavior.

downloadError

If download stops or fails for some reason this property will hold and error object with the reason for the failure.

downloadFinishedTime

Download Finish time. This is when the item was first moved to VOCItemState::VOCItemCached.

downloadQueuedTime

Download Queued time. This is when the item was first moved to VOCItemState::VOCItemQueued.

downloadStartTime

Download Start time. This is when the item was first moved to VOCItemState::VOCItemDownloading.

file

A content item can have multiple content url associated with it. Only one URL is selected for download and file property represents the file that URL downloads.

localPath

The local path to access the file represented by file property. This will nil if the content item is not cached.

locked

Item's locked stated.

paused

Paused item is that item whose download has been paused explicitly by the user. Paused item can be resumed to download only by the user. Paused item can still be evicted by the purging mechanism.

preferredQuality

Each file associated with the item might have a quality and the preferredQuality decides which file to download based on quality.

prioritizedDownload

Mark item to be downloaded as soon as possible.

priority

During ingest, the content item can be designated with priority. The download queue is sorted in descending order based on the priority.

size

A content item can have multiple content url associated with it. size property represents the size associated with the url that is selected for downloading.

source

The provider/source of the content item.

state

The primary aim for the SDK is to cache the item. This property indicates the downloading state of the item. For example: discovered, queued, downloading, idle, cached, failed etc.

summary

Summary of the content item.

thumbnail

Most of the media content (eg: video, audio) has a thumbnail associated. This property points to the thumbnail file associated with the content item.

title

Title of the content item.

uniqueId

Unique ID generated for the content by the PCD Server

url

A content item can have multiple content url associated with it. url property represents the url that is selected for downloading.
