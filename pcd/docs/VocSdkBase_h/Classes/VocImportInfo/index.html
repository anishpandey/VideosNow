VocImportInfo

Superclass:
NSObject
Declared In:
VocSdkBase.h

Introduction

For migrating a content into sdk, a VocImportInfo is passed into VocService::addNewContent::completion:.


Properties

contentDuration

This property expects the duration of the media file. @required For all migration process.

contentId

This value is the Content Id which client application uses for each content Item

contentPath

This property expects path to the main file. ie, For HLS, this will be the location of the StreamIndex manifest file. For MP4, this points to the MP4 media file.
